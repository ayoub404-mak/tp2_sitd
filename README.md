# TP N°2 : Programmation Orientée Objet LST SITD
Ce deuxième TP a pour objectif de consolider les notions fondamentales de la programmation orientée objet (POO) en Java.
À travers plusieurs exercices progressifs, l’étudiant doit être capable
de :
— Mettre en œuvre les principes d’abstraction et d’encapsulation ;
— Manipuler les constructeurs (par défaut, paramétré et de copie) ;
— Comprendre la notion de masquage d’attributs et son impact sur les objets ;
— Comparer des objets à l’aide de equals() et redéfinir correctement hashCode() ;
— Utiliser efficacement les collections Java : ArrayList, Set, et Map ;
— Supprimer des doublons et comparer des collections d’objets.

Résultat attendu : Maîtriser la création et la manipulation d’objets complexes, leur encapsu-
lation, leur comparaison, et la gestion d’ensembles d’objets dans des structures de données adaptées.
